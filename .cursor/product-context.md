# Product Context

## Product
AI-assisted VPN + Proxy Platform.

## Value Proposition
Simple and stable anti-block connectivity with intelligent fallback and diagnostics.

## MVP Focus
1. Stable infrastructure and backend first.
2. WireGuard primary path.
3. XRay fallback and anti-block path.
4. User onboarding and support via Telegram bot.
5. Rule-based smart recommendations and diagnostics.

## Primary User Outcomes
- Connect with minimum setup.
- Stay connected despite blocking events.
- Understand failures and get clear next actions.
